Protect Yourself 102

Take these simple steps to protect your devices, data, and identity while browsing online.

Use Strong Passwords

Create unique passwords with 12+ characters. Avoid using the same password across accounts.

Enable Two-Factor Authentication (2FA)

Add an extra layer of security by enabling 2FA via text, email, or an authenticator app.

Install Antivirus Software

Use reputable antivirus tools to detect and block malware and suspicious activity.

Keep Software Updated

Regularly update your apps, browser, and OS to patch security holes.

Avoid Public Wi-Fi for Sensitive Tasks

Use a VPN or mobile hotspot when checking banking or personal accounts.
